# Prediksi Waktu Terbaik Posting di Sosial Media Menggunakan Machine Learning

Projek ini menggunakan algoritma **Random Forest Regressor** untuk memprediksi waktu optimal posting di media sosial berdasarkan data engagement dari Kaggle.

## 📁 Struktur Folder Projek
* `data/` : Tempat menyimpan dataset mentah dan dataset bersih.
* `notebooks/` : Tempat menyimpan file Jupyter Notebook (.ipynb) untuk EDA dan visualisasi.
* `src/` : Tempat menyimpan skrip Python (.py) untuk preprocessing, training model, dan evaluasi.
